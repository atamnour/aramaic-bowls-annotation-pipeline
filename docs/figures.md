# Figures and Visual Assets

This folder contains lightweight visual assets used by the repository README and documentation.

## Assets

| File | Purpose |
|---|---|
| `docs/assets/pipeline_overview.png` | Main README banner showing the full extraction pipeline. |
| `docs/assets/original_bowl.png` | Original bowl sample. |
| `docs/assets/text_line_localization.png` | Text-line / baseline localization example. |
| `docs/assets/letter_detections.png` | Letter proposal detection example. |
| `docs/assets/sam_segmentation.png` | SAM-based segmentation refinement example. |

## Notes

- Keep these images small enough for GitHub rendering.
- Do not add restricted or full-resolution dataset images unless redistribution is allowed.
- Prefer PNG for diagrams and compressed JPEG/PNG for large visual examples.
